## About script
bash script to get IP Range (CIDR) at The Company <br>

## Requments
- amass tool
```console
snap install amass
```
- whois
```console
apt-get install whois
```
## Install
```console
svn checkout https://github.com/Abdulrahman-Kamel/mini-hacks/trunk/cidr
```

## Usage
```bash
sudo chmod +x cidr.sh
./cidr.sh yahoo results.txt
 ```
